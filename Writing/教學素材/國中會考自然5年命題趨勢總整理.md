# 國中會考自然 5 年命題趨勢總整理（110–114年）

> 綜合分析 110–114 年試題，涵蓋理化、生物、地科三大領域。  
> 詳細考題請見各科專項教學素材。

---

## 一、試卷結構

每年 **50–54 題**，三科分配大致固定：

| 科別 | 每年估計題數 | 比例 |
|------|------------|------|
| 理化（物理＋化學） | 18–22 題 | 約 40% |
| 生物 | 15–18 題 | 約 32% |
| 地科 | 13–16 題 | 約 28% |

**結論：理化題數最多，生物次之，地科最少但每題整合度高。**

---

## 二、5 年超高頻考點（每年必考）

### 理化（8 大主題）

| 主題 | 出現年份 | 代表考題 |
|------|---------|---------|
| 測量與實驗設計 | 110~114（5年） | 天平歸零/砝碼操作、溫度計懸吊位置 |
| 波動（聲/光） | 110~114（5年） | 共振條件、光色吸收、超聲波頻率 |
| 物質結構 | 110~114（5年） | 原子/離子電子數、元素週期表刪去法 |
| 化學反應 | 110~114（5年） | 酸鹼中和pH、氧化還原、有機化合物 |
| 力與運動 | 110~114（5年） | 慣性、F=ma、摩擦力、槓桿力矩 |
| 電與磁 | 110~114（5年） | 電鍍/電池、電功率計算、電路安全 |
| 溶液與濃度 | 110~114（5年） | 重量百分濃度、ppm、密度計算 |
| 反應速率 | 110~114（5年） | 接觸面積影響、動態平衡概念 |

### 生物（7 大主題）

## 三、近年命題趨勢變化（112–114 顯著）

| 趨勢 | 說明 | 影響備考方向 |
|------|------|------------|
| **跨域情境包裝** | 引用文學（小王子）、歷史（虎門銷煙）、時事（氫能分類）包裝自然題 | 要能先看懂情境再套用概念 |
| **圖表判讀強化** | 潮汐圖、氣溫預報圖、鋒面剖面圖、v-t關係圖大量出現 | 練習讀圖→計算一體的題型 |
| **科學方法比重上升** | 操縱變因、對照實驗設計明確考查，不再只考知識記憶 | 每科都要熟悉「控制變因辨析」 |
| **環境/能源議題** | 氫能分類（綠氫/灰氫）、碳排放、地層下陷、再生能源 | 新興議題需補充時事背景 |
| **整合型複合題** | 單題需同時判斷多個概念（如：生態+遺傳+演化合併出題） | 加強跨章節連結理解 |

## 四、各年度出題特色

| 年度 | 題數 | 特色 |
|------|------|------|
| 110 | 54題 | 實驗設計與生活情境（颱風、潮汐、觀星）並重 |
| 111 | 50題 | 遺傳（ABO血型三因子）、大氣壓應用（洋芋片）難度提升 |
| 112 | 42題+ | 時事議題首入題（氫能/碳捕捉）、皂化反應複合推理 |
| 113 | 47題+ | 跨文化素材（小王子/虎門銷煙）、果蠅遺傳複雜化 |
| 114 | 51題+ | 科學方法最強調年、鋒面與聲波情境包裝最深 |

## 五、備考重點建議

**最高 CP 值主題（必讀）**
1. 實驗設計的控制/操縱變因辨析（理化＋生物均考）
2. 遺傳：Aa×Aa→3:1、ABO血型（每年必考）
3. 鋒面：冷鋒/暖鋒特徵辨別（110~114均考）
4. 電功率計算：W=Pt、用電度數（每年理化必考）
5. 密度計算與三態變化（每年理化必考）

**近年新興必補**
- 氫能分類（綠氫概念）
- 科學記號（科學報告數值判讀）
- 生物保育IUCN分類概念
