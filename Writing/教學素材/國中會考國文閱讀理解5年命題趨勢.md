# 國中會考國文｜閱讀理解 5 年命題趨勢（110–114年）

> 閱讀理解佔每年 **30–38 題**，是整張試卷的核心。  
> 分為「古典詩文」與「現代詩文」兩大區塊，各含「單題」與「題組」。

---

## 一、題型結構與比例

### 閱讀理解四大區塊（每年固定）

| 區塊 | 說明 | 每年題數（估） |
|------|------|--------------|
| 古典詩文單題 | 文言文短段，單題各別問答 | 8–10 題 |
| 古典詩文題組 | 文言文長段或甲乙雙文，2–5 題為一組 | 4–6 題 |
| 現代詩文單題 | 現代文短段，單題各別問答 | 6–10 題 |
| 現代詩文題組 | 現代文長段、圖表或雙文，2–5 題為一組 | 12–18 題 |

> **趨勢：** 現代詩文題組比重逐年升高，111年起圖文整合題固定出現。

---

## 二、問題類型分析（★ = 高頻考法）

### ★★★ 主旨 / 觀點推論

> 問：「這段文字的主旨最接近下列何者？」、「最符合作者觀點？」

**答題口訣：找尾句、找轉折、看對比**

| 技巧 | 說明 | 5年例子 |
|------|------|--------|
| **找末句** | 文章結論通常在末段或末聯 | 110年Q29〈遊靈巖記〉「有其地而非其人…皆不足以盡夫遊觀之樂」 |
| **找轉折詞** | 「但/然而/卻/反而」後方通常是重點 | 110年Q12「被視為無用的…幸運逃過一劫，成了神木」→ 無用之用 |
| **找對比** | 兩種情況對比，結論偏向哪一邊 | 110年Q19 拉霸機研究：「現金→ 能察覺輸錢；卡片→ 難以察覺」 |
| **看問答結構** | 文章以問句開頭，答案在後半段 | 111年Q1 圖像知識→「重新組織，用自己的方式說出來」才是真知識 |

---

### ★★★ 細節 / 推論

> 問：「根據本文，下列何者最恰當？」、「可以推論出下列何者？」

**答題口訣：回文找依據，選項必有對應句**

| 常見陷阱類型 | 說明 | 例子 |
|------------|------|------|
| **過度推論** | 選項超出文章範圍 | 110年Q18「(A)海水雜質與水流速度無關」→ 文中明確說有關 |
| **正話反說** | 把肯定句改成否定、否定改肯定 | 113年Q9「(A)公子光接受餘祭按兄弟排行繼位」→ 原文說他接受的是（B）和（C） |
| **未提及** | 答案從文中找不到依據 | 111年Q25「(B)目的地是暖暖」→ 文中未說 |
| **數字陷阱** | 圖表題常在數字上設陷阱 | 112年Q21「(C)1900年衰退至1886年的50%」→ 120/240=50% ✓ |

---

### ★★★ 文意填詞（克漏字）

> 問：「根據文意脈絡，空格處填入下列何者最恰當？」

**答題口訣：空格前後都是線索，找邏輯關係**

| 技巧 | 說明 | 例子 |
|------|------|------|
| **前後關聯** | 空格前後句子的因果/轉折關係決定填什麼 | 110年Q19 空格後「所以…寧願放掉功課」→ 前句應填「等待報酬時間越短，動力越強」 |
| **動詞語感** | 填動詞時，注意詞語的程度與語氣 | 110年Q39 「風將帶著巨量的雨水□□過小城上空」→ 大風應用「狂奔」，非「輕掠」 |
| **代名詞指涉** | 注意「這/那/它/他」指的是什麼 | 111年Q35「孤絕丁零的餘數」指的是人（不是作品）|

---

### ★★ 寫作手法分析

> 問：「關於本文的寫作手法，下列何者最恰當？」

| 常考手法 | 說明 | 5年例子 |
|---------|------|--------|
| **先景後情／先情後景** | 詩歌上下片/聯的順序 | 110年Q25 辛棄疾詞「上下片先情後景」→ 實為先景抒感，答案為(B) |
| **對比（反襯）** | 兩種情況形成對比 | 112年Q12「老農與西江商賈對比，凸顯社會不公」 |
| **比喻** | 本體、喻體、喻詞 | 113年Q13「遺忘如同打燈後的暗處」 |
| **細筆（工筆）** | 詳細描寫細節 | 110年Q41 颱風散文「以細筆表現颱風將至到離去的種種景象」 |
| **象徵** | 物象代表某種情感或意義 | 112年Q26「以天氣的變化象徵越來越沉重的心情」→ 但文中心情先轉明再轉鬱，此選項錯 |

---

### ★★ 古典文言字義推論

> 問：「句中『○』字的意義，與下列何者最接近？」

**答題口訣：代入語境，不看表面字義**

| 高頻考字 | 在文中的意思 | 易混淆的意思 |
|---------|------------|------------|
| **坐** | 因此受罰（古義） | 坐下（現代義） |
| **信** | 確實、誠然 | 相信（現代義） |
| **亡** | 失去、逃亡 | 死亡（現代義） |
| **矯** | 改變、糾正 | 矯健（現代義） |
| **名** | 功名 | 姓名（現代義） |
| **比來** | 最近、近來 | 相比（現代義） |
| **一旦** | 忽然有一天 | 一旦（假設） |
| **令郎** | 敬稱對方的兒子 | 令（命令） |
| **俗故** | 日常俗務、世俗瑣事 | 老朋友 |
| **訓** | 典範、榜樣 | 訓誡（現代義） |

---

### ★★ 雙文對讀（甲乙文比較）

> 問：「根據甲乙兩文，下列何者最恰當？」、「哪個議題是兩文的歧見？」

**答題口訣：先確認兩文各自的主張，再找分歧點**

| 年份 | 甲乙文主題 | 歧見所在 |
|------|-----------|---------|
| 110 | 《夢溪筆談》（沈括）vs.《澠水燕談錄》（宋祁）關於柳開張景 | 兩人應試時間是否相近 |
| 112 | 月球大使館（甲）vs. 外太空條約（乙） | 私人是否可主張月球土地所有權 |
| 113 | 呂祖謙〈讀史論〉（甲）vs.《左宗棠全集》〈讀古人書〉（乙） | 對讀書方法的不同側重 |
| 114 | 對聯貼法甲圖 vs. 乙圖 | 是否皆說明仄起平收原則（皆是） |

### 現代文本題材五年演變

```
110年：人文散文 + 社會科普（饑荒/茶文化）
111年：圖表開始出現 + 哲學/心理/生活
112年：圖文整合主力登場 + 科技社會議題（隱私/月球）
113年：雙文對讀成標準格式 + 生活溝通/歷史
114年：圖表推理（足球賽程）+ 文學研究型說明文
```

**關鍵趨勢：**
1. **圖文整合越來越複雜**：從圖示（111）→ 計算型圖文（112跳水）→ 推理型賽程表（114）
2. **雙文對讀固定化**：111年甲乙文開始穩定出現，113年為讀書方法論高峰
3. **生活情境融入**：詐騙識別（114）、隱私保護（112）、問卷設計（113）
4. **說明文文體多元**：科普→心理→歷史藝術→文學研究，取材面越來越廣

## 五、圖文整合題深度解析（★★★ 近年最重要）

### 5 年圖表類題型盤點

| 年份 | 題號 | 圖表類型 | 核心技能 |
|------|------|---------|---------|
| 110 | Q16 | 國小新住民子女人數統計表 | 數據趨勢判讀（「漸趨下降」判斷） |
| 111 | Q11 | 背包重量建議圖示 | 條件對照（體重%門檻） |
| 111 | Q25 | 國人主要死因統計（107/97對照） | 跨年比較推論 |
| 112 | Q27-29 | 跳水比賽規則圖文 | 規則理解→數字計算（7+8+8）×3.0 |
| 112 | Q21 | 甲國茶葉出口歐洲統計表 | 圖表細節推論（衰退幅度計算） |
| 113 | Q6（圖） | 書店開幕廣告文案 | 廣告文本判讀 |
| 113 | Q17 | 性別平等統計圖（家暴通報） | 圖表+文字整合推論 |
| 114 | Q6（疫情圖） | 疫情與家暴通報圖 | 圖表細節精確判讀 |
| 114 | Q9 | 花茶種類對照表 | 條件對照（有無茶葉/有無花瓣） |
| 114 | Q21 | 世界盃足球賽程表 | 邏輯推理（淘汰制賽程） |

### 圖文整合解題策略

**Step 1：先讀題目，再看圖**  
→ 知道要找什麼數據，避免被多餘資訊干擾

**Step 2：找圖表的「說明文字」**  
→ 說明文字通常含有限制條件，是設陷阱的地方

**Step 3：逐項比對選項與圖表**  
→ 不要憑印象，每個選項都要找到圖表依據

**Step 4：小心「全稱/特稱」陷阱**
- 「全球女性」≠「曾有過伴侶的15至49歲女性」（111年Q17）
- 「近二分之一的篇章」≠「全書的二分之一」（114年Q15）

## 六、雙文對讀解題策略（★★）

### 解題流程

```
第一步：各自讀懂甲文主張
第二步：各自讀懂乙文主張
第三步：找兩文的「共同點」和「不同點」
第四步：按題目問的是「共同點」還是「歧見」作答
```

### 常考題型

| 題型 | 問法 | 解法 |
|------|------|------|
| 歧見所在 | 「哪個議題是甲乙歧見？」 | 找兩文針鋒相對的論點 |
| 共同說明 | 「甲乙皆說明了何者？」 | 找兩文都提到的內容 |
| 歸納推論 | 「根據甲乙，推論何者合理？」 | 兩文合併推論，不能只用一文 |
| 可信度比較 | 「哪文較可信？為何？」 | 對照外部資料（如丙表年份） |

## 七、核心解題策略總整理

### 策略一：答案一定在文章裡

> 閱讀理解不考「文章外的知識」，每個選項都能回原文找到或排除依據。

- 找不到依據 = 選項可能是「未提及」→ 通常是錯的
- 選項說「越來越……」→ 回去看圖表/數字確認趨勢方向

### 策略二：注意「主角轉換」

| 原文主角 | 選項誤寫 | 例子 |
|---------|---------|-----|
| 令尹怪罪江乙 | 誤寫為「江母告發令尹」 | 113年Q42 |
| 柳開推薦張景 | 誤寫為「王起推薦張景」 | 111年 |
| 私人探險隊達成目標 | 誤寫為「官方達成目標」 | 113年Q9 |

### 策略三：詩歌分析「先確定體裁」

```
看句數 → 4句=絕句，8句=律詩，長短不一=詞/散曲
看韻腳 → 偶句哪些字押韻？
看對仗 → 律詩第3、4句（頷聯）是否對仗？
```

### 策略四：文言文「先讀語譯，再看題目」

> 試卷通常附有語譯（語譯欄），先讀語譯能大幅提升理解速度，再細讀原文找字義依據。

### 策略五：克漏字「找邏輯詞」

| 邏輯詞 | 代表關係 | 空格填什麼 |
|--------|---------|---------|
| 所以、因此 | 因果 | 填「原因」（空格在前）或「結果」（空格在後） |
| 但、然而、卻 | 轉折 | 填與後句「相反」的內容 |
| 越…越… | 遞進 | 填符合遞進邏輯的詞 |
| 不僅…而且 | 遞進 | 後填程度更深的詞 |

## 八、5 年閱讀理解命題方向總結

### 穩定不變的考法

1. **每年必考「文意推論」**：根據文章細節推論，答案必有對應句
2. **每年必考「古典字義」**：坐/信/亡/矯等古今異義字
3. **每年必考「主旨推論」**：從末句/轉折詞找主旨
4. **每年必考「詩歌分析」**：體裁判斷+情感/手法分析

### 持續強化的趨勢（111年起）

1. **圖文整合越來越硬**：從圖示判讀→計算型→推理型
2. **雙文對讀固定出現**：每年至少一組甲乙文題組
3. **跨域取材**：科學/法律/心理/歷史藝術/文學研究皆入題

### 教學建議優先級

| 優先級 | 教學重點 | 理由 |
|--------|---------|------|
| ★★★ | 細節推論（回文找依據） | 佔最多題數 |
| ★★★ | 圖文整合（先讀說明、逐項比對） | 難度高，分差題 |
| ★★★ | 主旨推論（末句/轉折詞） | 每年必考 |
| ★★ | 雙文對讀（確認各文主張再比較） | 近年趨勢 |
| ★★ | 古典字義（代入語境） | 穩定考點 |
| ★ | 寫作手法（景情/比喻/對比） | 較難，少量出現 |
