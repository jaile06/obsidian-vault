## 四、go + Ving 活動（114年重點）

| 英文 | 中文 | 說明 |
|------|------|------|
| go dancing | 去跳舞 | 114，因為怕水而不選 |
| go hiking | 去健行 | 114 |
| **go sailing** | 去帆船 | 114，與水有關，本題答案 |
| go shopping | 去購物 | 114 |

> **文法提示**：go + Ving 是固定片語，表示「去做某種娛樂活動」，sailing（帆船）是水上活動，因此怕水的人不會選。

## 十二、數量代名詞辨析（常見陷阱）

| 英文 | 中文 | 使用情境 |
|------|------|---------|
| **another** | 另一個（不特定） | 還有很多個中的「又一個」 |
| **the other** | 另外那一個（特定） | 只剩最後一個 |
| one | 一個（不特定） | 泛指某一個 |
| both | 兩個都 | 恰好兩個 |
| all | 所有的 | 全部 |
| most | 大多數 | 不是全部 |
| some | 一些 | 部分 |
| each | 每一個 | 個別考量 |

> **考題陷阱**：另外那一個女兒用 **the other**（知道總共三個，兩個提過，剩最後一個）；錯過的公車用 **one**（不特定的一班）。

## 教學使用建議

1. **情緒形容詞**：最值得製作情境配對卡（圖片 + 情境句 → 選形容詞）
2. **片語辨析**：proud of / sorry for / worried about 三組放一起對比，效果最好
3. **介係詞看圖**：直接使用歷屆第一題圖片，培養圖文對應能力
4. **數量代名詞**：another vs. the other 是每年必考的迷惑選項，需要反覆練習
