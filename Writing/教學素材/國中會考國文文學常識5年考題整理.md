# 國中會考國文｜文學常識 5 年考題整理（110–114年）

> 資料來源：110–114 年國中教育會考國文科試題解析（翰林版）  
> 文學常識涵蓋：漢字形體演變、六書造字原則、詩詞格律、應用文題辭。  
> 每年約出現 **3–5 題**，集中在語文常識與古典詩文分析區塊。

## 一、漢字形體演變

### 各年考題

| 年份 | 題號 | 考點 | 正確答案 |
|------|------|------|---------|
| 111 | Q4 | 「獸」字四種字體排序 | (B) 金文→小篆→隸書→楷書 |
| 113 | Q10 | 〈瑯琊臺刻石〉為何種字體 | (A) 篆書（用筆圓轉、筆畫粗細均勻） |
| 114 | Q5 | 《說文解字》收錄哪些字體 | (A) 楷書**不**收錄（以小篆為主，兼古文字與籀文） |

---

### 五種字體特徵對照表（★ = 高頻考點）

| 字體 | 時代 | 視覺特徵 | 關鍵詞 |
|------|------|---------|--------|
| **甲骨文** | 商代 | 刻在龜甲獸骨，線條細瘦，象形性強 | 占卜、契刻 |
| **金文** ★ | 商周 | 鑄在青銅器，字形結構未固定，象形字仍有圖畫性質 | 銘文、肥厚 |
| **籀文** | 周代 | 線條圓勻、粗細不一，屬秦以前古文字 | 《說文》收錄 |
| **小篆** ★ | 秦代 | 筆畫圓勻**粗細一致**，結構整齊 | 統一文字、李斯 |
| **隸書** ★ | 漢代 | 由圓轉變為**平直**，筆畫尾部有**波勢挑法**（蠶頭雁尾） | 波磔、隸變 |
| **楷書** ★ | 東漢起 | 去掉隸書波勢挑法，**筆畫平直、結構方正**，通行至今 | 正楷、端正 |

> **辨別口訣：** 甲→金（圖畫味）→ 小篆（圓勻統一）→ 隸（波勢挑法）→ 楷（平直方正）

---

### 《說文解字》小常識（114年考點）

- 作者：東漢 **許慎**
- 收錄字體：**小篆**為主，兼採秦以前古文字（含籀文）
- **楷書未收錄**（楷書約完備於曹魏，晚於許慎）

---

### 四大造字原則（★ = 高頻，每年必考）

#### ★ 象形（最常出現）
> 描繪具體實物外形造字

| 例字 | 說明 |
|------|------|
| **月** | 外部象月亮邊緣，內部象陰影 |
| **木** | 上象樹枝，中象樹幹，下象樹根 |
| **齒**（甲骨文） | 象口中牙齒形狀 |
| **雨** | 象天空降雨形狀 |

#### ★ 指事（抽象符號）
> 在象形字上附加**抽象符號**，表示抽象概念

| 例字 | 說明 |
|------|------|
| **本** | 木（象形）＋一（符號標根部） |
| **小** | 甲骨文用三點或四點表示細小之義 |
| **上、下** | 基準線上下加一點 |

#### ★ 會意（合體表義）
> 兩個或以上的形符組合，會合其義

| 例字 | 說明 |
|------|------|
| **刪** | 刀＋冊（用刀刮削竹簡） |
| **出** | 止（腳）＋凵（穴居住所）→ 出門離去 |
| **炎** | 火＋火（火上加火，更熱） |

#### ★ 形聲（形符＋聲符）
> 一部分表義（形符），一部分表音（聲符），佔漢字最大比例

| 例字 | 形符（義） | 聲符（音） |
|------|-----------|-----------|
| **汝** | 水（與水相關） | 女（音ㄖㄨˇ） |
| **齒**（小篆） | 齒形（義） | 止（音） |
| **勁** | 力 | 工 |
| **燈** | 火 | 登 |

> **記憶口訣：**「象形描物、指事抽象、會意合義、形聲最廣」

### 絕句 vs. 律詩 辨別表（★★★ 每年必考）

| 項目 | 絕句 | 律詩 |
|------|------|------|
| 句數 | **4 句** | **8 句** |
| 字數 | 五言或七言 | 五言或七言 |
| 用韻 | 偶句押韻（第2、4句），首句可押可不押 | 偶句押韻（2、4、6、8句） |
| 對仗 | **不要求** | **頷聯（3、4句）、頸聯（5、6句）必須對仗** |
| 常考辨別點 | 4句、無強制對仗 | 頷頸聯對仗工整 |

> **常見陷阱：** 出題時常在「這是七言律詩」的選項中，把沒有對仗的詩設為正確答案的陷阱，考生要確認**頷聯（第3、4句）有無對仗**。

---

### 詞的格律要點（★★）

| 項目 | 說明 | 常見誤區 |
|------|------|---------|
| **詞牌** | 規定音樂格式（如西江月、鷓鴣天）| 詞牌 ≠ 題目 |
| **題目** | 另加的說明文字 | 「江行采石岸」是題目，「西江月」是詞牌 |
| **上下片（闋）** | 詞分上片（前段）+ 下片（後段） | |
| **韻腳** | 詞可換韻（與律詩不同） | |
| **對仗** | 部分詞牌要求某些句子對仗 | 如〈西江月〉上下片首二句皆對仗 |

---

### 散曲的格律要點（★，114年新考）

| 項目 | 說明 |
|------|------|
| **曲牌** | 散曲的音樂格式（如小梁州） |
| **題目** | 曲牌後另加的說明（如「秋」） |
| **小令** | 散曲中僅一支曲子（單支曲） |
| **一韻到底** | 散曲從頭到尾押同一個韻（不換韻）|
| **韻腳範例** | 〈小梁州〉秋：黃、光、藏、盪、香、望、江、漾、行（陽韻） |

## 五、對聯格律

### 114年 Q7 考點整理

| 規則 | 說明 |
|------|------|
| **仄起平收** | 上聯末字為**仄聲**（上/去/入），下聯末字為**平聲** |
| **貼的位置** | 面對大門：**右邊貼上聯**，左邊貼下聯 |
| **貼的順序** | 上聯 → 下聯 → 橫批 |
| **對仗要求** | 對應位置詞性相同，平仄相對 |

### 教學重點排序

1. **六書造字原則**（象形/指事/會意/形聲）→ 5 年全考，優先施教
2. **漢字字體特徵**（金文/小篆/隸書/楷書）→ 4 年出現
3. **應用文題辭**（醫/教/商/競賽場合配對）→ 4 年出現
4. **絕句 vs. 律詩辨別**（句數/對仗/用韻）→ 3 年出現
5. **詞的格律**（詞牌/上下片）→ 3 年出現
6. **散曲**（曲牌/小令/一韻到底）→ 114年新考點，預計持續
