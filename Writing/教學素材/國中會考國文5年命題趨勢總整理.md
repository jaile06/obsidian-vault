# 國中會考國文 5 年命題趨勢總整理（110–114年）

> 綜合分析 110–114 年試題，含字音字形、文學常識、文法、閱讀理解四大面向。  
> 詳細考題請見各專項教學素材。

---

## 一、試卷結構

每年 **40–48 題**，配分結構固定：

| 大類 | 題數 | 比例 |
|------|------|------|
| 語文基本能力（字音字形、詞語、標點、語法） | 5–7 | 約 15% |
| 語文常識（六書、應用文、文體格律） | 3–5 | 約 10% |
| 閱讀理解（古典＋現代，單題＋題組） | 30–38 | 約 75% |

**結論：閱讀理解佔 3/4，是決定成績的關鍵。**

---

## 二、語文基本能力（字音字形）

每年固定：**字音 1 題 + 字形 1 題**，共 2 分，難度不高但陷阱固定。

**字音題規律：**
- 多音字必出干擾（咽/吭/禪/撒）
- 罕見字考正確讀音：垠(ㄧㄣˊ)、罄(ㄑㄧㄥˋ)、揶(ㄧㄝˊ)、藩(ㄈㄢˊ)
- 正確答案多為「形近字同音組」：囑/矚、抵/邸

**字形題規律（112 年起）：**
- 必考成語錯字：趕盡殺絕（進→盡）、情不自禁（盡→禁）、眼不見為淨（靜→淨）
- 形近字：索→瑣、默→陌、盃→碑

---

## 三、語文常識

每年 3–5 題，三個核心考點：

**① 六書造字原則（5 年全考，最重要）**

| 造字法 | 定義 | 高頻例字 |
|--------|------|---------|
| 象形 | 描繪具體實物 | 月、木、齒（甲骨文） |
| 指事 | 象形＋抽象符號 | 本（木＋一）、小（三點） |
| 會意 | 兩形符合義 | 刪（刀＋冊）、出（止＋凵） |
| 形聲 | 形符＋聲符 | 汝（水＋女）、燈（火＋登） |

**② 漢字字體演變（4 年出現）**
- 金文→小篆→隸書→楷書，各字體特徵必記
- 《說文解字》收小篆與籀文，**不收楷書**（114 年考點）

**③ 應用文題辭（4 年出現）**

| 場合 | 正確題辭 |
|------|---------|
| 醫院／醫師 | 杏林之光、仁心仁術 |
| 教育／老師 | 桃李芬芳、春風化雨 |
| 書法比賽 | 鐵畫銀鉤 |
| 歌唱比賽 | 新鶯出谷 |
| 演說比賽 | 口若懸河 |
| 作文比賽 | 妙筆生花 |

## 四、文法（語言使用）

每年 3–5 題，三類必考：

**① 連詞句型（110 年 Q1，也是其他年干擾項）**

| 配對 | 關係 |
|------|------|
| 即使…也 | 讓步 |
| 與其…不如 | 選擇（✗ 與其…難道 = 語病） |
| 只有／除非…才 | 條件 |
| 無論／不管…都 | 無條件 |

**② 成語使用（每年必考，2 題以上）**

高頻錯誤：
- **對象錯**：汗牛充棟形容書，不能形容教室；車水馬龍形容車馬，不能形容船
- **褒貶錯**：前倨後恭（貶，勢利）≠ 謙和有禮；急公好義（褒）≠ 莽撞
- **語意矛盾**：不言而喻（大家都懂）接「大家都不知道」→ 矛盾

**③ 謙詞／敬詞（113 年重點）**
- 謙詞只能**自己用**：淺見、薄酌、獻醜、拙作
- 敬詞只能**稱對方**：高見、蒞臨、令郎、令尊

---

## 五、閱讀理解（最核心，75%）

分四區塊，現代文本題組最難、分差最大。

**5 年現代文本取材演進：**

```
110 → 人文散文 + 社會科普（饑荒／茶文化）
111 → 圖表初登場 + 哲學／心理／生活
112 → 圖文整合成主力（跳水計算）+ 科技社會（隱私／月球）
113 → 雙文對讀固定化 + 生活情境（溝通／遠征）
114 → 推理型圖表（足球賽程）+ 文學研究型說明文
```

**古典文本固定類型：**
宋詞格律分析、莊列寓言、筆記小品（《夢溪筆談》《世說新語》）、史傳人物、唐詩鑑賞。

---

## 六、三大命題趨勢（111 年起加速）

**趨勢 1：圖文整合越來越難**
- 111：圖示判讀（背包重量門檻）
- 112：計算型（跳水評分 (7+8+8)×3.0）
- 114：邏輯推理（淘汰制賽程，誰一定贏誰）

**趨勢 2：雙文對讀固定出現**
- 每年至少一組甲乙文（4–5 題）
- 核心技巧：先各自理解兩文主張，再找分歧點

**趨勢 3：生活情境與跨域題材**
- 詐騙識別、數位隱私、問卷設計、疫情家暴、月球法律
- 不再只考「純文學」，需要廣泛閱讀各類文章

---

## 七、解題策略優先順序

| 優先 | 策略 |
|------|------|
| 1 | **答案一定在文章裡**——找不到依據的選項排除 |
| 2 | **主旨看末句**——文章結論通常在最後一句或一段 |
| 3 | **轉折後是重點**——「但／然而／卻」後方是作者真正的意思 |
| 4 | **圖表先讀說明**——說明文字有限制條件，是陷阱所在 |
| 5 | **雙文先各自理解**——再比對，不要混在一起讀 |
| 6 | **克漏字找邏輯詞**——「所以」前填原因、「但」後填反義 |

```
  > 圖文整合
  > 主旨推論
  > 六書造字原則

```
