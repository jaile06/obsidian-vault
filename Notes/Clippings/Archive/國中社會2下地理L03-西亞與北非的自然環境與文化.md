# 一、西亞與北非的地形

- 1. 部分西亞與北非地區多位於【*1*板塊接觸帶 】上,地形深受影響。
- 2. 西亞地形:以【*2*高原 】為主。

| 區域 | 地形區                          | 成因與特徵              |  |

| 北  | 伊朗高原、安納托力亞高原                 | 受板塊擠壓抬升較晚,地震多且地勢較高 |  |

| 南  | 阿拉伯高原                        | 地質古老,高原面較平坦、少高山    |  |

- (1) 分布著廣大的撒哈拉沙漠。

#### 三、伊斯蘭教的教義

- 1. 教義明確,只要做到以下,即可獲得救贖:
  - (1) 從日常生活中嚴守【<sup>5</sup>五功】: 念、禮(每日須朝【<sup>6</sup>麥加】做五次禮拜)、齋、課、朝。
  - (2) 定期到【7清真寺 】禮拜。
- 2. 伊斯蘭教經典《【<sup>8</sup>古蘭經】》雖有不同語言的翻譯本,但須使用阿拉伯文本念誦才有意義, 無形中促進**伊斯蘭世界**的形成。

#### 四、生活方式

- 1. 伊斯蘭教在西亞與北非的信仰比例較高。
- 2. 此區氣候乾燥,生活方式可見適應氣候與宗教信仰的巧思:
