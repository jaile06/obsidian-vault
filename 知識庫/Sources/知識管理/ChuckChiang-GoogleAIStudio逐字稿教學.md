## 摘要（3 句內）

Chuck Chiang 示範使用 Google AI Studio（免費）搭配 Gemini 2.5 Pro，只需貼上 YouTube 網址並輸入「提取影片逐字稿」，即可在數十秒內取得無字幕影片的逐字稿。逐字稿可直接複製貼入 NotebookLM，解決 NotebookLM 無法匯入無字幕影片的限制。

## 關鍵要點

- 操作路徑：Google AI Studio → Chat 模式 → Gemini 2.5 Pro → 加號 → YouTube Video → 貼網址 → Add to Prompt → 「提取影片逐字稿」
- 免費，每日 token 額度足夠處理 20 分鐘以內影片
- 輸出後：三個點點 → Copy → 貼至 NotebookLM「貼上文字」
