## 摘要

介紹如何在 20 分鐘內用 Obsidian + Claudian 插件打造 AI 知識庫 Agent（4.0 版），核心是將 AI 從「被動工具」升級為「主動管家」——透過 skill 指令系統自動捕獲、整理、回顧知識，並將知識庫作為驅動創作工作流的引擎。

## 關鍵要點

### 4.0 版知識庫核心升級

- AI 從「被問才答」→「主動整理歸類、發現連結、生成報告」
- 6 個核心資料夾：日常工作 / 工作專項 / 專業領域 / 個人思考 / AI學習實操 / 日常記錄（萬能收件箱）

### Claudian Skill 指令系統

| 指令 | 功能 |
|------|------|
| `/ask` | 搜尋知識庫筆記 |
| `/report` | 自動生成週報 |
| `/review` | 定期回顧摘要 |
| `/daily` | 生成每日任務（含昨日未完成事項） |
| `/capture` | 快速捕獲網頁或文字 → 結構化筆記 |

### 知識複利理念

- 個性化數據是 AI 時代最寶貴的數位資產
- 你喂給 AI 的數據越個性化，AI 就越懂你
- 知識庫不只是筆記本，更是 AI 的記憶系統與訓練數據
- 長期積累形成「知識複利」：今天的靈感 × 3 個月後的筆記 = 新想法

### 工作流閉環

輸入（隨手記錄） → 沉澱（Agent 整理歸類）→ 輸出（選題報告 + 視頻大綱）
