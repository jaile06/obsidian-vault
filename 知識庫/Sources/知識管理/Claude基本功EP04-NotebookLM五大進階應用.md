## 摘要（3 句內）

Claude Code 透過 MCP 串接 NotebookLM，可突破網頁版限制，實現自動化、定時化、行動化、串聯化、規模化五大應用。一句話 Skill 可觸發備課完整流程（上傳資料→簡報→影片→資訊圖表→差異化教材→測驗），跨工具整合 NotebookLM、Obsidian、GitHub。200+ 本筆記本可批次盤點、分類、重新命名，解決網頁版無法大規模管理的痛點。

## 關鍵要點

### 五大「化」應用

- **自動化**：用 [[Claude-Skill]] 將備課流程（上傳 → 簡報、影片、音訊、資訊圖表、差異化教材、測驗）打包成一句話觸發
- **定時化**：用 Claude Schedule 設定每週自動執行 deep research，將報告下載至本地資料夾
- **行動化**：用 Claude Dispatch 從手機遠端操控 Claude Code，指令同步至電腦端執行
- **串聯化**：跨工具一句話整合 — NotebookLM 報告 → Obsidian 筆記 + 簡報 → GitHub 備份
- **規模化**：批次盤點 200+ 本筆記本，分析使用頻率、命名規則重整

### MCP 使用建議

- NotebookLM 的操作花 Google 帳號額度；下載/轉檔花 Claude 額度
- [[MCP]] 佔 Context Window 空間，只裝工作最需要的幾個，不要亂裝

### Skill 建立流程

1. 先手動完整跑一遍流程，確認 ok
2. 請 Claude 將流程打包成 Skill（檔案型 Skill，存在全域 skill 資料夾）
3. 之後只需提及觸發詞，自動執行所有步驟
