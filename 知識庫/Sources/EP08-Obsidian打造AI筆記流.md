## 摘要（3 句內）

示範以 Karpathy 筆記原則結合 Obsidian + Claude Code 建立三層資料結構的第二大腦。透過 Web Clipper 抓取文章與 NotebookLM 抓取逐字稿，原始資料不動、由 AI 每週自動消化進知識庫。設有每週自動排程（週日 9:17）執行七步驟知識重整，產出週報。

## 關鍵要點

- **三層資料結構**：Clipping（外部原始）/ 創作庫（自創原始）/ 知識庫（AI整理）
- 原始資料**永遠不修改**，知識庫由 AI 管理
- **claude.md**：告訴 AI 該如何整理，每次啟動自動讀取
- **NotebookLM 整合**：可抓取無字幕音訊逐字稿（8 萬字）
- 每週知識重整七步驟：盤點 → 消化 Clipping → 提煉創作 → Lint → 更新 Index/Log → 健康檢查 → 週報
- 知識庫支援 Graph View，AI 自動建立跨文章連結
- 問答結果可回流知識庫，越用越豐富
