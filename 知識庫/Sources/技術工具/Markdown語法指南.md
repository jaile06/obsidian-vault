## 摘要（3 句內）

Markdown 是輕量級標記語言，透過簡單符號實現標題、清單、表格、程式碼等常用排版。支援語法高亮：在程式碼區塊標記語言名稱（如 ```php）即可啟用。大多數平台（GitHub、GitLab、Obsidian）原生支援 Markdown 渲染。

## 關鍵要點

- 標題：`#`（一至六層）
- 粗體/斜體：`**粗體**` / `*斜體*`
- 列表：`-` 無序、`1.` 有序
- 連結：`[文字](URL)`；圖片：`![描述](網址)`
- 引用：`>`
- 程式碼：行內用反引號，區塊用三個反引號 + 語言名稱
- 表格：`| 欄位 | 欄位 |` + `|---|---|`
- PHP 高亮：` ```php ` + PHP 代碼 + ` ``` `
