## 摘要（3 句內）

以 Adobe Color 從圖片抓取 5 色調色盤，並搭配 ChatGPT Prompt 逐步完成色彩角色定義、和諧檢查、對比度測試，最終輸出 CSS 色彩變數與導覽列、區塊的完整配色方案。涵蓋從 HEX/RGBA/HSLA 格式下載，到套用主色、輔色、強調色、背景色、文字色的標準流程。適合需要快速為網站建立一套可維護配色系統的開發者。

## 關鍵要點

- **取色步驟**：上傳圖片到 Adobe Color → 下載 CSS（HEX / RGBA / HSLA）→ 貼入 ChatGPT
- **ChatGPT Prompt 順序**：
  1. 色彩角色定義（主色/輔色/強調色/背景色/文字色）
  2. 色彩和諧 + 灰階對比檢查
  3. 對比度測試（文字 vs 背景 ≥ 4.5:1）
  4. 版面顏色分配（左側選單/主內容/右側資訊欄）
  5. 導覽列完整互動配色（文字、底色、hover 狀態、子選單）
  6. 區塊（Section/Card）色彩風格（含 text-shadow）
- **輸出格式**：HEX → CSS；RGBA → 設計軟體；HSLA → 漸層
