## 摘要（3 句內）

Claude Design 是 Anthropic 推出的視覺設計工具，底層由 Opus 4.7 驅動，使用者可用自然語言建立品牌設計系統，並一站式生成 PPT、網站原型等設計作品。與 Lovable 等前端 AI 工具相比，Claude Design 強調視覺回饋與品牌一致性，支援截圖、Figma 檔、HTML 代碼等多種輸入方式。作者實測結論：one-shot 完成度已能威脅基礎設計工作，「會用 AI 才是關鍵」。

## 關鍵要點

- **品牌設計系統（Design System）**：先設定品牌調性（色票、字體、風格），後續所有設計自動保持一致
- **三種輸入方式**：截圖、Figma 檔、HTML Extractor 抓取的網站代碼
- **支援輸出類型**：Prototype、Slide Deck、Landing Page、App 原型
- **Tweaks 功能**：生成後可在界面直接用自然語言微調局部設計
- **模型**：Opus 4.7（地表最強視覺模型）
- **與 Claude Code 整合**：設計完成後可無縫交接給 Claude Code 本地執行
- **Cloudflare 限制**：若網站有 Cloudflare 保護，HTML 抓取可能失敗，需改用 URL 直接輸入
