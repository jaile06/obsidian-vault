Markdown 語法是一種輕量級的標記語言，主要用於格式化文本，方便在網頁或文檔中顯示排版效果。這裡是一些基本的 Markdown 語法介紹：

### 1. 標題（Headings）

使用 `#` 來表示標題，數量越多代表標題層級越低。

```markdown
# 這是 H1 標題
## 這是 H2 標題
### 這是 H3 標題
#### 這是 H4 標題
##### 這是 H5 標題
###### 這是 H6 標題
```

### 2. 粗體和斜體

- 粗體：使用 `**` 或 `__` 包裹文字。
    
- 斜體：使用 `*` 或 `_` 包裹文字。
    

```markdown
**這是粗體文字**
__這也是粗體文字__

*這是斜體文字*
_這也是斜體文字_
```

### 3. 列表

#### 無序列表

使用 `*`、`+` 或 `-` 創建無序列表。

```markdown
* 項目 1
* 項目 2
  * 子項目 2.1
  * 子項目 2.2
```

#### 有序列表

使用數字加點（例如 `1.`）來創建有序列表。

```markdown
1. 第一項
2. 第二項
3. 第三項
```

### 4. 鏈接和圖片

#### 鏈接

使用 `[顯示文字](URL)` 格式來插入鏈接。

```markdown
[Google](https://www.google.com)
```

#### 圖片

在鏈接前加上 `!` 符號來插入圖片。

```markdown
![圖片描述](圖片網址)
```

### 5. 引用

使用 `>` 來創建引用。

```markdown
> 這是一段引用文字。
```

### 6. 程式碼

#### 行內程式碼

使用單個反引號（`` ` ``）包裹程式碼。

```markdown
這是 `程式碼` 的例子。
```

#### 區塊程式碼

使用三個反引號（` ``` `）來創建程式碼區塊。

這是一個程式碼區塊。

### 7. 水平線

使用三個以上的 `-`、`*` 或 `_` 創建水平線。

```markdown
---
```

### 8. 表格

使用 `|` 和 `-` 創建表格。

```markdown
| 標題 1 | 標題 2 | 標題 3 |
|--------|--------|--------|
| 單元格 1 | 單元格 2 | 單元格 3 |
| 單元格 4 | 單元格 5 | 單元格 6 |
```

### 9. 插入HTML

- 如果 Markdown 語法無法實現某些功能，可以直接插入 HTML 代碼。

```markdown
<p>這是 HTML 代碼</p>
```
