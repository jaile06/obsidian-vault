## 🔑 圖片 → 網站配色工作流

### ① 上傳圖片並取色

- **工具**：Adobe Color、Coolors、Canva Color Palette
    
- **步驟**：
    
    1. 上傳圖片
        
    2. 選擇「色彩風格」(鮮豔、柔和、深沉、亮色、暗色)
        
    3. 下載 HEX / RGB 色碼
        

👉 得到 **3–6 個主色**（一般建議 5 色）。

---

### ② 定義角色（色彩職責）

- **主色 (Primary)** → 品牌色、網站/海報主調
    
- **輔色 (Secondary)** → 配合主色，避免單調
    
- **強調色 (Accent)** → 用於按鈕、標題、重要提醒
    
- **背景色 (Background)** → 頁面底色，通常淺色系
    
- **文字色 (Text)** → 主要字體顏色，深灰或黑，確保對比度
    

👉 例子：

- 主色：#3A7CA5
    
- 輔色：#F2B880
    
- 強調色：#D94F30
    
- 背景：#F9F9F9
    
- 文字：#222222
    

---

### ③ 色彩和諧檢查

- **工具**：Adobe Color → 色彩規則 (互補、三分、類似色)
    
- **Tips**：
    
    - 避免所有顏色同等強烈 → 需要有「主角 + 配角」
        
    - 用 **灰階測試**：把配色轉灰階，檢查是否仍有對比
        

---

### ④ 對比度測試（無障礙檢查）

- **工具**：WebAIM Contrast Checker / Contrast Ratio
    
- **原則**：
    
    - 文字 vs 背景 對比 ≥ 4.5:1（小字）
        
    - 大字 ≥ 3:1
        
    - 按鈕/連結一定要「背景明顯、hover 狀態更清楚」
        

---

### ⑤ 應用到設計樣式

- **網站**：
    
    - `Primary` → 按鈕主色、標題字
        
    - `Secondary` → 卡片背景、區塊色條
        
    - `Accent` → 重點標籤、CTA 按鈕 hover
        
    - `Background` → 頁面主底
        
    - `Text` → 預設文字
        
- **簡報/海報**：
    
    - `Primary` → 主標題
        
    - `Secondary` → 插圖或圖表填色
        
    - `Accent` → 強調重點文字或箭頭
        
    - `Background` → 投影片底色
        

---

### ⑥ 輸出與保存

- **輸出格式**：
    
    - HEX（#FF5733） → 網頁 / CSS
        
    - RGB (255, 87, 51) → 設計軟體
        
    - HSL (10°, 100%, 60%) → 色彩漸層
        
- **保存方式**：
    
    - 存成 **.ase (Adobe Swatch Exchange)** 或 **Coolors PDF**
        
    - 建議再做一份 **配色指南 (Style Guide)**，方便團隊使用
