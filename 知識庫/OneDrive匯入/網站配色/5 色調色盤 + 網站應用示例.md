請幫我做流程檢查：
##定義角色（色彩職責）
- **主色 (Primary)** → 品牌色、網站/海報主調    
- **輔色 (Secondary)** → 配合主色，避免單調    
- **強調色 (Accent)** → 用於按鈕、標題、重要提醒    
- **背景色 (Background)** → 頁面底色，通常淺色系    
- **文字色 (Text)** → 主要字體顏色，深灰或黑，確保對比度
##色彩和諧檢查
- 避免所有顏色同等強烈 → 需要有「主角 + 配角」    
- 用 **灰階測試**：把配色轉灰階，檢查是否仍有對比
##對比度測試（無障礙檢查）
- **原則**：
    
    - 文字 vs 背景 對比 ≥ 4.5:1（小字）        
    - 大字 ≥ 3:1        
    - 按鈕/連結一定要「背景明顯、hover 狀態更清楚」
##應用到設計樣式
- **網站**：
    
    - `Primary` → 按鈕主色、標題字
        
    - `Secondary` → 卡片背景、區塊色條
        
    - `Accent` → 重點標籤、CTA 按鈕 hover
        
    - `Background` → 頁面主底
        
    - `Text` → 預設文字
        
- **簡報/海報**：
    
    - `Primary` → 主標題
        
    - `Secondary` → 插圖或圖表填色
        
    - `Accent` → 強調重點文字或箭頭
        
    - `Background` → 投影片底色
