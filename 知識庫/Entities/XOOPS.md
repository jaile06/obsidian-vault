## 簡介

XOOPS（eXtensible Object Oriented Portal System）是以 PHP 開發的開源模組化 CMS 平台，廣泛用於社群網站、機關入口網站與教育平台。本知識庫主要使用版本為 **XOOPS 2.5**。

## 主要觀點與貢獻

- 模組化架構：功能以模組（module）為單位獨立開發，可自由安裝／移除
- 2.5 版使用 Procedural PHP + Smarty 模板引擎
- 內建 `$xoopsDB` 資料庫抽象層與 Token 驗證機制
- 本知識庫針對 2.5 版有一套完整的 [[XOOPS開發規範]]
