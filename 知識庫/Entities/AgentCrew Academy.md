## 簡介

Dustin 主持的 Claude Code 實戰教學頻道，以四層設定系統（CLAUDE.md / Rules / Memory / Hooks）解說如何讓 Claude 真正「記住你的規則」。

## 主要觀點與貢獻

- **四層設定系統**：CLAUDE.md（默契檔）→ Rules（條件載入）→ Memory（Claude 自寫動態狀態）→ Hooks（程式層強制執行）
- **Context Window 節省策略**：CLAUDE.md 控制在 200 行以下，超出的規則移至 Rules 按路徑按需載入
- **Hook 優先原則**：對於「忘了就風險大」的規則，從自然語言建議升級為程式碼強制攔截
