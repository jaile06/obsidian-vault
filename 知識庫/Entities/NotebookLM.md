## 簡介

Google 開發的 AI 筆記工具，以 RAG 架構為核心，適合整理大量文字資料並進行問答。

## 主要觀點與貢獻

- 底層使用 [[RAG]] 技術，回答基於使用者上傳的文件
- 適合處理文字多的資料（如學術論文、長篇報告）
- 可生成逐字稿摘要、問答互動

## Claude Code 串接後的五大應用（[[MCP]]）

透過 [[MCP]] 將 Claude Code 與 NotebookLM 串接，可突破網頁版限制：

- **自動化**：[[Claude-Skill]] 打包備課流程（上傳 → 簡報 / 影片 / 資訊圖表 / 差異化教材 / 測驗）
- **定時化**：排程每週自動 deep research，將報告下載至本地
- **行動化**：手機 Claude Dispatch 遠端下指令，電腦端自動執行
- **串聯化**：一句話跨三工具 — NotebookLM → Obsidian → GitHub
- **規模化**：批次盤點 / 分類 / 重新命名 200+ 本筆記本

## 與其他工具的關聯

- 可搭配 Claude Code 使用（[[MCP]] 協議）
- 與 [[ChatGPT]] 定位不同，偏向「資料整理」而非「通用對話」
- 串聯 [[第二大腦]]（Obsidian）與 GitHub 形成完整知識流
