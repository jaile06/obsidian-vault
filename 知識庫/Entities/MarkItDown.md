## 簡介

Microsoft 開源 Python 套件，將各類型檔案（PDF、Word、PPT、Excel、圖片、音訊等）轉換為 Markdown，設計目標為 LLM 與文字分析 pipeline 的前置處理工具。

## 主要觀點與貢獻

- 支援格式廣泛（PDF / Office / 圖片 / 音訊 / YouTube URL）
- Markdown 輸出保留文件結構（標題、清單、表格、連結）
- 提供 CLI、Python API、Docker 三種使用方式
- 插件生態：`markitdown-ocr` 支援以 LLM Vision 做圖片 OCR
