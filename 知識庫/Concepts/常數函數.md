## 定義

函數通式 y = b（b 為常數），斜率為 0，圖形為平行 x 軸的水平線。

## 核心重點

- **通式**：y = b
- **斜率**：0（無傾斜）
- **截距**：y 軸截距 = b，無 x 軸截距（除非 b = 0）
- **圖形**：水平直線

## 與線型函數的關係

- [[線型函數]] 通式 y = ax + b：當 a = 0 時，退化為常數函數
- [[一次函數]]：a ≠ 0，不包含常數函數

## 範例

| 函數 | 圖形 |
|------|------|
| y = 3 | 水平線，y 固定為 3 |
| y = -2 | 水平線，y 固定為 -2 |
| y = 0 | x 軸本身 |
