## 定義

在設計系統中為每種顏色指定明確職責，確保視覺層級清晰、用途不混亂。

## 核心重點

| 角色 | 用途 |
|------|------|
| 主色 (Primary) | 品牌色、按鈕主色、主標題 |
| 輔色 (Secondary) | 配合主色，卡片背景、區塊色條 |
| 強調色 (Accent) | 重點標籤、CTA 按鈕、hover 狀態 |
| 背景色 (Background) | 頁面底色，通常淺色系 |
| 文字色 (Text) | 主要字體，深灰或黑，確保對比度 |

## 不同視角 / 矛盾之處

- 部分設計系統會區分 Primary / Primary-variant（深淺版），讓同色系有更多彈性

## 延伸問題

- 功能性語意色（紅色=錯誤、綠色=成功）是否算第六種角色？
