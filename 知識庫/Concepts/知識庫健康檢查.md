## 定義

定期（每週）自動掃描知識庫，檢查孤立頁面、斷裂連結、內容矛盾、過薄頁面等問題，輸出健康報告。戴倫以 `/vault-lint` 自訂 skill 實踐。

## 檢查項目（10 項）

1. 孤立頁面（無任何連結指向）
2. 斷裂的 `[[wikilink]]`（連結目標不存在）
3. 缺少 frontmatter 的頁面
4. 過薄頁面（內容 < 3 行）
5. 重複概念（相似名稱的 Concept）
6. Sources 無對應 Concepts 提取
7. Concepts 無 Sources 支撐
8. 長期未更新的頁面
9. Index.md 與實際檔案不一致
10. 知識缺口（頻繁提及但無獨立頁面的詞）

## 輸出格式

- 每週一 08:00 自動執行（遠端 CCR）
- 產出 `WeeklyReport-YYYY-MM-DD.md`
- 更新 `log.md`
- git commit + push 回 GitHub
