## 定義

Obsidian 內建 PDF 檢視器，加上 PDF to Markdown 外掛，可將 PDF 文件的文字內容一鍵轉為 Markdown，直接進入筆記庫進行後續整理。

## 核心重點

### 內建 PDF 檢視器（Obsidian 0.10.8+）

- 無需外掛，原生支援
- 可捲動、指定起始頁碼、選取與複製文字

### PDF to Markdown 外掛

- 一鍵將整份 PDF 轉為 Markdown 文字
- 安裝：【設定】→【第三方外掛】→【瀏覽】→ 搜尋 `PDF to Markdown` → 安裝 → 啟用
- 需求：Obsidian 安裝程式版本 0.8.x 以上

## 應用場景

| 情境 | 做法 |
|------|------|
| 論文、報告擷取重點 | PDF to Markdown 轉換後，AI 協助摘要 |
| 掃描文件數位化 | 轉為 Markdown 後可搜尋、標記 |
| 配合自動入庫 | 轉換後的內容接入 AI 整理流程 |
